# URL for an example of an LP DAAC COG file

URL for an example of an LP DAAC COG file

## Usage

``` r
lpdacc_example_url()
```

## Value

The URL to a Cloud-Optimized Geotiff file from the LP DAAC.

## Examples

``` r
lpdacc_example_url()
#> [1] "https://data.lpdaac.earthdatacloud.nasa.gov/lp-prod-protected/HLSL30.020/HLS.L30.T56JKT.2023246T235950.v2.0/HLS.L30.T56JKT.2023246T235950.v2.0.SAA.tif"
```
